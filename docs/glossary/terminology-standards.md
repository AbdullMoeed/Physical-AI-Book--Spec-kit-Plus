# Terminology Standards

![Standardized Terminology](https://images.unsplash.com/photo-1543286386-713bdd548da4?ixlib=rb-4.0.3&ixid=M3wxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8fA%3D%3D&auto=format&fit=crop&w=2370&q=80)

This document defines the standard terminology used throughout the Physical AI Book curriculum. Consistent use of terminology helps ensure clear communication and understanding. Key standards include:

- Using "Physical AI" to refer to AI systems interacting with the physical world
- Distinguishing between "sensing" (input) and "actuation" (output)
- Standardizing component names and technical terms
- Consistent formatting of code and technical references

Following these standards will enhance your learning experience.
