# Hardware Guide

![Hardware Components for Physical AI](/img/hardware-components-diagram.svg)

This guide provides information about the hardware components used throughout the Physical AI Book curriculum. Common components include:

- Microcontrollers (Arduino, Raspberry Pi, ESP32)
- Sensors (temperature, light, motion, etc.)
- Actuators (motors, LEDs, displays)
- Communication modules (WiFi, Bluetooth)
- Power supplies and connectors

This guide details specifications, connections, and use cases for each component.
