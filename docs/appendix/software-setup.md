# Software Setup

![Software Stack for Physical AI](https://images.unsplash.com/photo-1555066931-4365d14bab8c?ixlib=rb-4.0.3&ixid=M3wxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8fA%3D%3D&auto=format&fit=crop&w=2370&q=80)

This guide provides instructions for setting up the software tools used throughout the Physical AI Book curriculum. Essential tools include:

- Text editor or IDE (recommended: VS Code)
- Python or Node.js runtime
- Git for version control
- Specific libraries and frameworks
- Development environments for hardware platforms

Proper setup is crucial for success with the hands-on activities.
